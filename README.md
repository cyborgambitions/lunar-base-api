A sleek, futuristic web dashboard for monitoring the **space economy** with a strong focus on the **lunar market**. Built to support Moon Alpha Base strategic planning, investment tracking, and real-time market intelligence.

## Overview

The **Space Economy Market Dashboard** provides real-time market data, portfolio simulation tools, AI-powered predictions, and curated news for companies shaping the future of lunar commerce and in-space infrastructure.

This project serves as the digital command center for tracking public and private players in the emerging space economy — including direct support for **SpaceX** impact analysis and **Lockheed Martin**.

## Key Features

- **Real-time Market Data** — Live prices and performance for key space stocks and ETFs
- **Company Intelligence Endpoints** — Dedicated data for Lockheed Martin, SpaceX (proxy), Rocket Lab, and more
- **Interactive Portfolio Simulator** — Simulate positions and projected returns
- **Futuristic UI/UX** — Dark, neon-accented, space-themed interface
- **Lunar Economy Predictions** — Forward-looking insights tied to Moon Alpha Base development
- **Curated News Feed** — News and events that directly impact lunar markets
- **Modular Architecture** — Clean separation between backend APIs and frontend
